# question-images

Imagens de apoio de questões, extraídas dos PDFs das provas e servidas
publicamente (mesmo modelo das imagens do enem.dev já usadas no banco).

Cada arquivo é `<question_id>.png` e é referenciado por `questions.image_url`
via URL raw do GitHub. **Não apague nem renomeie** estes arquivos: eles são
exibidos para os alunos ao fazer os simulados. Removê-los quebra a imagem da
questão correspondente.

Backfill inicial: simulados "Especial de férias" e "Professores - Humanas".
